# Component Garden

## Description

Component Garden is a React playground (garden) showcasing my knowledge and practice of React Hooks, Reusable Components, and State Management (including Zustand).

## Live Demo

[View live site](https://component-garden.netlify.app/)

## Technologies Used

- React
- React Router
- Zustand

## Features

- Custom reusable UI components
- Global state management with Zustand
- React Hooks practice (useEffect, useMemo, custom hooks, etc.)
- Routing with React Router
- Component catalogue/playground UI
